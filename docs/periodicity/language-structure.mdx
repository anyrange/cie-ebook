# Language structure

## Comparative and Superlatives
<dl>
  <dt><b>all – все (используется)</b></dt>
  <dd><em>e.g.</em> <strong>All elements </strong>are located in the Periodic table according to their atomic mass.</dd>
  <dt><b>adjective + er … than… (односложные прилагательные)</b></dt>
  <dd><em>e.g.</em> Sodium atom has a <strong>lower</strong> atomic radius <strong><u>than</u></strong> the atom of potassium.</dd>
  <dt><b>more + adjective – (двух и более сложные прилагательные)</b></dt>
  <dd><em>e.g.</em> Fluorine is <strong>more reactive</strong> <strong><u>than</u></strong> iodine.</dd>
  <dt><b>the + adjective + est</b></dt>
  <dd><em>e.g.</em> Tungsten has <strong>the highest</strong> melting point.</dd>
  <dt><b>the + most + adjective</b></dt>
  <dd><em>e.g.</em> Francium has <strong>the most striking</strong> metallic properties.</dd>
  <dt><b>as… + adjective + … as –</b></dt>
  <dd><em>e.g.</em> Sodium is <strong>as good</strong> an electric conductor <strong>as</strong> potassium.</dd>
  <dt><b>the more … + adjective + the more …</b></dt>
  <dd><em>e.g.</em> <strong>The higher</strong> the number of the period where an element is located <strong>the bigger</strong> its radius</dd>
</dl>
