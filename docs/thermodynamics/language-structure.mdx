# Language structure

## CONDITIONALS (REVIEW)

<center>
  If + <s>WILL</s>
</center>

<table>
  <tr>
    <th>If clause</th>
    <th>Main clause</th>
  </tr>
  <tr>
    <td>If ∆G equals to -ve,</td>
    <td>the reaction will proceed.</td>
  </tr>
  <tr>
    <th>If + Present Simple,</th>
    <th>Will + Verb</th>
  </tr>
</table>

_e.g._ If ∆G **is equal to** +ve, the reaction will not proceed, even if ∆H = -ve.  
_e.g._ If ∆H **is** –ve and ∆S is positive, the reaction **will be** spontaneous at all temperatures.  
_e.g._ If ∆H **is** +ve and ∆S is negative, the reaction **will not be** spontaneous at any temperature.  
_e.g._ If ∆H and ∆S **are** both +ve, the reaction **will be** spontaneous only above a certain temperature.
