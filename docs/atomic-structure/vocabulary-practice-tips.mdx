### Labelling a diagram

A  labelled  diagram  is  a drawing or a chart that is used to visualize a concept. This  task  is  essential  for visual  learners.  Labelling diagram  can  be  used  to help students understand the  relationships  among different  parts  of  the diagram. A CLIL teacher can use it to learn new subject-specific vocabulary.
